# shadcn-ui

开发环境中已经集成了 shadcn-ui，如需使用 shadcn-ui 组件，需要在对应的代码文件中引入，示例如下

```typescript
import { Alert } from './components/ui/alert';
```

环境中可用的 shadcn-ui 组件包括：
accordion, alert, alert-dialog, aspect-ratio, avatar, badge, breadcrumb, button, calendar, card, carousel, checkbox, collapsible, command, dialog, drawer, form, input, label, pagination, popover, progress, radio-group, select, separator, sheet, skeleton, slider, sonner, switch, table, tabs, textarea, toast, toaster, tooltip
