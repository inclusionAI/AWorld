# FullCalendar

开发环境中已经集成了 FullCalendar，如需使用 FullCalendar 组件，需要在对应的代码文件中引入，示例如下

```typescript
import FullCalendar from '@fullcalendar/react';
import dayGridPlugin from '@fullcalendar/daygrid';
```

环境中可用的 FullCalendar 包包括：
@fullcalendar/core, @fullcalendar/daygrid, @fullcalendar/react
