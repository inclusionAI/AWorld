# App Generation Recipe

Generate, evaluate, and self-evolve a complete mini-application from a single text prompt. The AWorld Agent orchestrates the full flow: from your idea to a polished, functional mini-app with optional quality targets.

## What You Get

- A runnable mini-application (e.g. web app) generated from your description
- Automatic evaluation and self-evolution to meet quality criteria you specify
- One command from prompt to product

## Example: English Word Learning Mini-App

**Prompt you can use:**

```text
Help me create an English word learning mini-app with a UI quality score above 0.9.
```

Replace the goal (e.g. “English word learning”) and the quality target (e.g. “UI quality score above 0.9”) with your own. The agent will generate the app, evaluate it, and iterate until the criteria are met.

## How to Run

1. Install and configure AWorld-CLI (see the main [README](../../README.md)).
2. Launch the CLI and choose or enter the app-generation flow.
3. Paste or type your prompt (e.g. the one above).
4. Let the agent generate, evaluate, and evolve the mini-app; then run or deploy the output as needed.

For installation and activation details, see [Your Journey with AWorld-CLI](../../README.md#your-journey-with-aworld-cli) in the README.
